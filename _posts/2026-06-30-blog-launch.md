## 这是什么

一个自动发布的 GitHub 博客。我有心得输入给 Hermes Agent，它帮我格式化、排版、推送到 GitHub Pages——不需要登录 GitHub、不需要手动操作。

## 怎么工作

```
我的心得 → Hermes Agent → Markdown格式化解码 → git push → GitHub Pages → 博客上线
```

整个流程一句话触发：**「帮我把这个写成博客发上去」**。

## 解码格式

每篇文章遵循四步解码框架：

1. **发现问题** —— 切开的具体问题，之前没人解决什么？
2. **骨架认识** —— 因果链 + 核心变量
3. **点 → 线 → 面** —— 关键数据 → 逻辑链条 → 更大图景
4. **落到我身上** —— 今天能做的精确动作

## 关于我

「大山里的解码者」—— 在代码与论文之间架桥，把学术黑话拆解成人话。
